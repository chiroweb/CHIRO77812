## 홈페이지 비용은 제작비로 끝나지 않는다

홈페이지 제작비를 300만원 냈다고 해서 이후 비용이 없는 것은 아닙니다. 사이트가 온라인에 있는 한 매달, 또는 매년 발생하는 비용이 있습니다.

이 비용을 미리 파악하지 않으면 예상치 못한 청구서를 받거나, 유지보수 계약을 과도하게 비싼 금액에 맺는 일이 생깁니다.

항목별로 실제 비용을 정리합니다. 빌더 기반과 커스텀 코딩 기반을 나눠서 비교합니다.

---

## 항목 1: 호스팅 비용

사이트 파일을 저장하고 인터넷에 공개하는 서버 비용입니다. 사이트가 존재하는 한 반드시 발생합니다.

**빌더(아임웹, 카페24, Wix)**

빌더는 플랫폼 구독료에 호스팅이 포함됩니다. 가격은 다음과 같습니다.

- 아임웹: 월 1만 7천원~6만 6천원 (플랜에 따라 다름)
- 카페24: 기본 무료이지만 독자 도메인, SSL, 고급 기능은 월 1만~5만원
- Wix: 월 약 2만 5천원~7만원 (USD 기준, 환율 적용)

연간으로 환산하면 20만~80만원 사이입니다.

**커스텀 코딩(Vercel, AWS, Netlify)**

- Vercel 무료 플랜: 소규모 사이트는 무료 사용 가능 (트래픽 제한 있음)
- Vercel Pro: 월 약 20달러(약 2만 7천원)
- AWS S3 + CloudFront: 트래픽에 따라 월 1천~1만원 수준
- Netlify 무료 플랜: 월 100GB 대역폭까지 무료

커스텀 코딩 기반 사이트는 대부분 월 1만~3만원 수준으로 운영 가능합니다. 빌더보다 호스팅 비용이 낮은 경우가 많습니다.

---

## 항목 2: 도메인 비용

.com, .co.kr 같은 도메인 주소 비용입니다. 연간 단위로 갱신합니다.

- .com: 연 1만 5천~2만원 (가비아, 후이즈 기준)
- .co.kr: 연 2만 2천원 내외
- .kr: 연 1만 8천원 내외

도메인은 제작사가 대신 구매해주는 경우가 있지만, 반드시 본인 명의로 등록해야 합니다. 제작사 명의 도메인은 관계가 끊어질 경우 문제가 될 수 있습니다.

---

## 항목 3: SSL 인증서 비용

사이트 주소 앞에 https://가 붙게 해주는 보안 인증서입니다. 없으면 브라우저에서 "안전하지 않음" 경고가 표시됩니다. 구글은 SSL이 없는 사이트를 검색 순위에서 불이익을 줍니다.

**빌더**: 대부분 구독료에 포함. 별도 비용 없음.

**커스텀 코딩**
- Let's Encrypt: 무료. Vercel, Netlify, Cloudflare는 자동으로 무료 SSL을 제공합니다.
- 유료 SSL: 연 5만~50만원. 금융, 쇼핑몰처럼 강화된 보안이 필요한 경우에 한해 사용합니다.

일반 기업 홈페이지라면 무료 SSL로 충분합니다.

---

## 항목 4: 보안 업데이트 비용

사이트에 사용된 라이브러리, 플러그인, CMS 소프트웨어는 보안 취약점이 발견될 경우 업데이트가 필요합니다.

**빌더**: 플랫폼이 자동으로 처리합니다. 별도 비용 없음.

**WordPress 기반 커스텀 사이트**: 플러그인과 테마 업데이트를 직접 관리해야 합니다. 유지보수 계약을 맺는 경우 월 5만~20만원 수준의 비용이 발생합니다. WordPress는 보안 취약점이 많아 정기적인 관리가 필요합니다.

**Next.js, React 기반 커스텀 사이트**: 서버리스 구조로 운영하면 보안 관리 부담이 낮습니다. Vercel 기반은 인프라 보안을 Vercel이 담당합니다. 월 별도 보안 비용이 거의 발생하지 않습니다.

치로가 Next.js 기반으로 만드는 이유 중 하나입니다. WordPress 대비 보안 유지 비용이 낮습니다.

---

## 항목 5: 콘텐츠 관리 비용

블로그 글 작성, 포트폴리오 업데이트, 서비스 내용 수정, 이미지 교체 같은 작업 비용입니다.

직접 관리 가능 여부에 따라 비용이 달라집니다. CMS가 연결된 사이트는 관리자 페이지에서 직접 수정할 수 있습니다. CMS가 없으면 수정 요청마다 제작사에 비용을 지불해야 합니다.

일반적인 수정 대행 비용은 건당 2만~10만원 수준입니다. 월 3~4건이라면 연간 100만원 이상이 됩니다.

---

## 항목 6: 유지보수 계약 비용

제작사와 월간 유지보수 계약을 맺는 경우 발생하는 비용입니다.

유지보수 계약의 범위는 업체마다 다릅니다. 일반적으로 포함되는 항목은 다음과 같습니다.

- 정기적인 백업
- 보안 점검
- 소규모 수정 대응 (월 n시간 이내)
- 오류 대응 및 사이트 다운 시 복구

비용은 월 10만~50만원 수준입니다. 계약 전에 무엇이 포함되는지를 명확히 확인해야 합니다. "유지보수 포함"이라고 되어 있어도 실제로는 대형 수정은 별도 비용인 경우가 많습니다.

## 빌더가 장기적으로 더 비싼 이유

빌더의 월 구독료는 낮아 보이지만, 5년 기준으로 계산하면 다릅니다.

아임웹 기본 플랜(월 2만 5천원) 기준으로 5년이면 150만원입니다. 같은 기간 Vercel Pro를 사용하면 월 2만 7천원, 5년에 162만원으로 비슷합니다.

하지만 빌더는 플랜을 업그레이드할수록 비용이 올라갑니다. 트래픽이 늘거나 기능이 필요하면 상위 플랜으로 이동해야 합니다. 커스텀 코딩은 트래픽이 늘어도 비용 구조가 거의 바뀌지 않습니다.

더 중요한 것은 기회비용입니다. 빌더 구독료를 5년간 내면서 검색 유입이 없는 것과, 커스텀 코딩으로 검색 유입을 확보하는 것의 차이는 비용 비교를 의미 없게 만들 만큼 큽니다.

## 유지보수 비용을 최소화하는 방법

유지보수 비용을 낮추려면 처음부터 올바른 구조로 만들어야 합니다.

CMS를 연결해서 콘텐츠는 직접 관리하고, 기술적 유지보수만 외부에 맡기면 비용이 낮아집니다. 보안 업데이트가 자동화되는 플랫폼(Vercel, Netlify)을 사용하면 별도 보안 비용이 거의 없습니다.

치로는 Vercel 기반의 Next.js 사이트를 납품합니다. 납품 후 호스팅은 고객 Vercel 계정에 설정하고, 기본 수정이 가능한 CMS를 연결합니다. 별도 유지보수 계약 없이도 사이트를 운영할 수 있는 구조입니다.

[치로 가격 안내](/pricing)에서 제작 후 운영 비용 구조를 확인할 수 있습니다.

제작을 검토하기 전에 현재 사이트의 유지보수 구조가 합리적인지 확인하고 싶다면 [치로에 문의](/contact)해주시면 됩니다.

## 자주 묻는 질문

**유지보수 계약을 맺지 않으면 사이트가 망가지나요?**
반드시 그렇지는 않습니다. 정적 사이트(Next.js, HTML 기반)는 외부 공격에 강하고 자동으로 깨지지 않습니다. WordPress 기반은 플러그인 취약점 때문에 정기 관리가 필요합니다. 어떤 기술 스택으로 만들었는지에 따라 다릅니다.

**Vercel 무료 플랜으로 운영해도 되나요?**
소규모 기업 홈페이지는 무료 플랜으로 충분한 경우가 많습니다. 월 방문자가 1만 명을 넘거나 서버 기능(API)을 많이 사용하면 Pro 플랜(월 약 2만 7천원)으로 전환하면 됩니다.

**도메인을 제작사가 구매해줬는데, 소유권이 내 것인가요?**
등록자 명의를 확인해야 합니다. 가비아나 후이즈 같은 도메인 등록 업체에서 로그인해보면 확인됩니다. 제작사 명의로 되어 있다면 즉시 본인 명의로 이전을 요청해야 합니다. 도메인은 브랜드 자산입니다.

**아임웹을 해지하면 사이트 데이터를 가져올 수 있나요?**
텍스트 콘텐츠는 일부 내보내기가 가능하지만, 디자인 구조와 코드는 가져올 수 없습니다. 빌더에서 커스텀 코딩으로 이전할 때는 콘텐츠는 옮기고, 디자인과 코드는 새로 제작합니다.
