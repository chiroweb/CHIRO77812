## 전환 비용은 신규 제작과 거의 같습니다

아임웹에서 커스텀 코드 기반 사이트로 전환하는 비용은 신규 사이트 제작 비용의 90~100% 수준입니다. "이미 있는 것을 옮기는 것"이라 더 쌀 것 같지만, 코드 전체를 새로 작성해야 하기 때문입니다. 비용이 비슷하다면, 처음부터 제대로 만드는 편이 낫습니다.

---

## 전환 4단계와 소요 시간

### 1단계: 콘텐츠 추출 (1~2일)

기존 사이트의 텍스트, 이미지, 페이지 구조를 추출합니다. 아임웹은 콘텐츠 내보내기 기능이 제한적이라 수작업 비중이 높습니다. 이미지는 원본 파일이 없으면 재촬영이 필요한 경우도 있습니다.

### 2단계: 신규 사이트 구축 (7~10일)

콘텐츠를 바탕으로 커스텀 코드로 사이트를 제작합니다. 이 단계에서 시맨틱 HTML, 구조화 데이터, Core Web Vitals 최적화가 함께 적용됩니다.

### 3단계: SEO 자산 보존 설정 (1일)

이 단계가 핵심입니다. 기존 URL이 바뀌면 구글이 쌓아온 검색 자산이 사라집니다. 301 리디렉션을 적용하면 기존 주소로 들어오는 트래픽과 검색 점수가 새 주소로 이어집니다. 치로는 모든 이전 URL에 대해 리디렉션 맵을 작성합니다.

### 4단계: 도메인 전환 및 검증 (1일)

기존 도메인을 새 서버로 연결합니다. DNS 변경 후 24~48시간 내에 전 세계 서버에 반영됩니다. 전환 직후 Google Search Console에서 크롤링 오류 여부를 확인합니다.

---

## 기존 SEO 점수를 잃지 않으려면

전환 후 순위가 일시적으로 흔들리는 것은 정상입니다. 구글이 새 구조를 재평가하는 시간이 필요합니다. 그러나 301 리디렉션이 올바르게 적용됐다면 2~4주 내에 이전 수준을 회복하고, 구조 개선 효과로 더 오르는 경우가 대부분입니다.

반대로 리디렉션 없이 전환하면 기존에 쌓인 백링크, 색인, 순위가 전부 초기화됩니다.

## 자주 묻는 질문

**아임웹 구독료는 전환 후 바로 해지해도 되나요?**
새 사이트가 안정적으로 운영되는 것을 2~4주 확인한 후 해지하는 것을 권장합니다. 예상치 못한 문제 발생 시 원본을 참조할 수 있습니다.

**전환 중에 사이트가 다운되지 않나요?**
도메인 전환 전까지 기존 사이트는 그대로 운영됩니다. 새 사이트가 완성된 후 도메인만 연결하기 때문에 다운타임은 사실상 0입니다.
