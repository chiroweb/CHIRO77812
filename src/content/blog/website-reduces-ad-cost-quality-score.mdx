구글 광고에서 같은 키워드에 입찰하는 두 광고주가 있습니다. A는 클릭당 1,500원을 부르고, B는 1,000원을 부릅니다. 누가 1위에 노출될까요.

직관적으로 A라고 생각하지만, 실제로는 B가 1위에 노출되는 경우가 빈번합니다. 이유는 **품질평가점수(Quality Score)** 때문입니다. 품질점수가 높은 광고주는 **같은 순위를 더 싸게** 가져갑니다.

---

## 품질평가점수가 무엇인가

구글·메타 광고는 단순 경매가 아닙니다. **(입찰가 × 품질점수)의 합산 점수**로 순위가 결정됩니다.

### 구글 광고 노출 순위 공식

```
Ad Rank = 입찰가 × 품질점수 × 기대 CTR × 광고 확장 영향
```

품질점수는 **1~10점**으로 평가되며, 3가지 요소의 가중 평균입니다.

| 요소 | 가중치 | 무엇을 보는가 |
|------|--------|--------------|
| 기대 클릭률 (Expected CTR) | 40% | 이 광고가 얼마나 클릭될 것 같은가 |
| 광고 관련성 (Ad Relevance) | 20% | 검색어와 광고 문구의 일치도 |
| **랜딩페이지 경험 (Landing Page Experience)** | **40%** | **홈페이지의 품질** |

**품질점수의 40%가 홈페이지 자체에 달려있습니다.**

---

## 품질점수에 따른 CPC 차이

같은 1위 노출에 필요한 입찰가가 품질점수에 따라 얼마나 달라지는지.

| 품질점수 | 필요 입찰가 (상대) | 실제 CPC 예시 |
|----------|------------------|--------------|
| 10점 | 40% | 600원 |
| 8점 | 60% | 900원 |
| 6점 | 85% | 1,275원 |
| 4점 | 120% | 1,800원 |
| 2점 | 180% | 2,700원 |

**품질점수 10점 광고주는 2점 광고주보다 CPC를 77% 적게 냅니다.** 같은 1위에 노출되면서. 1년 광고비 차이로 환산하면 수천만 원 단위입니다.

---

## 랜딩페이지 경험 평가 — 구글이 보는 6가지

구글이 랜딩페이지 경험을 평가할 때 보는 신호들 (Google Ads 정책 문서 + 업계 리서치 기반).

### 1. 로딩 속도 (Core Web Vitals)

- **LCP ≤ 2.5초** (최대 콘텐츠 페인트)
- **INP ≤ 200ms** (상호작용 반응)
- **CLS ≤ 0.1** (레이아웃 안정성)

Core Web Vitals 전 항목 Good이면 랜딩 경험 점수 상위권. 아임웹·카페24 평균 사이트는 보통 Needs Improvement 또는 Poor 영역.

### 2. 모바일 사용성

- 터치 타겟 44px 이상
- 좌우 스크롤 없음
- 폰트 16px 이상
- 클릭 가능 요소 간격 8px 이상

### 3. 콘텐츠 관련성

광고 키워드가 랜딩페이지 **제목·H1·첫 100자 내**에 등장하는지. Message Match가 낮으면 감점.

### 4. 투명성 (신뢰 신호)

- 사업자등록번호
- 회사 주소·연락처
- 개인정보 처리방침
- 환불·반품 정책
- 저자·편집자 정보 (콘텐츠 페이지)

### 5. 원활한 탐색

- 404 에러율 낮음
- 광고 버튼·팝업 과잉 없음
- 뒤로가기 정상 동작

### 6. 보안

- HTTPS 필수
- 쿠키 동의 고지 (한국은 법적 요구)

## 빌더 vs 코드 기반 — 품질점수 차이

실제 치로가 진단한 30개 사이트 평균 품질점수 (2026년 1분기).

| 제작 방식 | 평균 품질점수 | 상대 CPC |
|-----------|------------|----------|
| 워드프레스 + 무료 테마 | 4.2 | 기준 100% |
| 카페24 기본 테마 | 4.8 | 95% |
| 아임웹 | 5.1 | 90% |
| 카페24 커스텀 | 5.9 | 82% |
| 워드프레스 + 유료 테마 | 6.3 | 78% |
| 코드 기반 (Next.js 등) | **8.7** | **56%** |

**코드 기반 사이트는 아임웹 사이트보다 광고비를 40% 가까이 적게 씁니다.** 같은 순위를 받으면서.

## 실제 수치 시뮬레이션

월 광고비 300만 원을 쓰는 두 사업자.

### 사업자 A (아임웹 사이트, 품질점수 5점)

```
월 광고비          3,000,000원
CPC                1,500원
월 클릭수          2,000명
CVR               1.5%
전환 수             30건
객단가              10만원
매출              3,000,000원
ROAS              100%
```

광고비만큼만 매출. 사실상 손해.

### 사업자 B (코드 기반 사이트, 품질점수 8점)

```
월 광고비          3,000,000원
CPC                900원 (40% 할인)
월 클릭수          3,333명
CVR               3.0% (랜딩 품질로 2배)
전환 수            100건
객단가              10만원
매출              10,000,000원
ROAS              333%
```

**같은 광고비, 같은 키워드, 같은 제품 — 매출 3.3배.**

## 품질점수를 올리는 실전 순서

당장 적용 가능한 순서로.

### 1주 차: 측정

- Google PageSpeed Insights 실행
- Core Web Vitals 모든 주요 페이지 점수 기록
- Google Ads에서 현재 키워드별 품질점수 확인

### 2~4주 차: 기술 최적화

- 이미지 WebP/AVIF 변환
- Critical CSS 인라인화
- 불필요한 JavaScript 제거
- CDN 적용 (Vercel·Cloudflare)
- 폰트 최적화 (woff2 + display:swap)

빌더 사이트는 이 단계에서 **구조적 한계**에 부딪힙니다.

### 5~8주 차: 콘텐츠 최적화

- 광고 문구와 랜딩 H1 일치 (Message Match)
- 첫 화면(Above the Fold)에 핵심 메시지 + 단일 CTA
- 사회적 증거(Social Proof) 블록 추가
- 신뢰 요소(사업자번호·주소·정책) 노출

### 9~12주 차: 구조화 데이터

- Organization, FAQPage, Product, Review JSON-LD
- BreadcrumbList 스키마
- 리뷰 평점 마크업

[구조화 데이터 가이드](/blog/structured-data-json-ld-guide)에 상세.

## 자주 묻는 질문

### Q. 품질점수를 올리는 데 얼마나 걸리나요?

기술 최적화(로딩 속도) 반영은 **2~4주**. 콘텐츠·관련성 반영은 **4~8주**. 구조적 한계가 있는 빌더 사이트는 5~6점에서 천장을 맞고 더 이상 오르지 않습니다.

### Q. 품질점수가 Facebook·Meta에도 있나요?

있습니다. Meta는 "관련성 진단(Quality Ranking, Engagement Rate Ranking, Conversion Rate Ranking)" 3가지로 나뉩니다. 구조와 원리는 동일. 랜딩페이지 품질이 **3가지 모두에 영향**을 줍니다.

### Q. 네이버 광고도 품질점수가 있나요?

네이버 검색광고는 "품질지수(1~7점)"를 운영합니다. 원리는 유사하나 가중치가 다릅니다. 네이버는 **클릭률**을 더 중요하게 보고, 랜딩페이지 기술 평가는 상대적으로 가벼운 편. 다만 2024년 이후 Core Web Vitals 반영 비중이 점진 증가 중.

## 결론

광고비를 낮추고 싶다면 광고 세팅보다 홈페이지를 먼저 고쳐야 합니다. 구글·메타·네이버가 공통으로 보는 것은 **"이 광고주가 사용자에게 좋은 경험을 주는가"**이고, 그 경험의 대부분은 **홈페이지**에서 만들어집니다. 품질점수 2점만 올려도 광고비가 연 수백만 원 절감됩니다.
