두 브랜드를 비교해 봅니다. 같은 뷰티 카테고리, 비슷한 제품, 비슷한 가격대.

- **브랜드 A**: 월 광고비 500만원 집행 → 월 매출 3,000만원. **광고 중단 → 다음 달 매출 300만원**.
- **브랜드 B**: 월 광고비 500만원 집행 → 월 매출 3,500만원. **광고 중단 → 다음 달 매출 2,200만원**.

매출은 비슷한데 광고 탄력성이 전혀 다릅니다. 차이를 만드는 변수는 5가지입니다.

---

## 두 구조의 근본 차이

### 광고 의존형 구조 (A)

```
매출의 90%가 광고 유입에서 발생
  ↓
광고 끄면 유입 0에 수렴
  ↓
다음 달 매출 10% 미만
```

### 혼합형 구조 (B)

```
매출의 35%가 광고 유입, 40% 오가닉, 15% 직접 유입, 10% 추천
  ↓
광고 끄면 유입 40%까지 감소
  ↓
다음 달 매출 60% 이상 유지
```

B 구조는 광고를 꺼도 사업이 멈추지 않습니다. 이것이 "**사업의 안정성**"의 실체입니다.

---

## 변수 1 — 브랜드 검색량

사람들이 당신의 브랜드명을 **직접 검색**하는 양.

| 월 브랜드 검색량 | 광고 탄력성 |
|----------------|------------|
| 0~200 | 광고 끄면 매출 95%↓ |
| 200~1,000 | 광고 끄면 매출 70%↓ |
| 1,000~5,000 | 광고 끄면 매출 40%↓ |
| 5,000~20,000 | 광고 끄면 매출 20%↓ |
| 20,000+ | 광고 끄면 매출 10% 이하↓ |

브랜드 검색량은 광고와 관계없이 들어오는 **가장 확실한 오가닉 유입**입니다. 광고를 6~12개월 꾸준히 돌리면 브랜드 검색량이 쌓입니다. 자사 도메인이 있어야 이 검색어의 1위를 가져갈 수 있습니다.

---

## 변수 2 — SEO 오가닉 유입 비율

롱테일 키워드·주제 권위로 들어오는 트래픽.

### 성숙한 사이트의 이상 비율

| 기간 | 오가닉 유입 비중 |
|------|-----------------|
| 런칭 0~3개월 | 5~10% |
| 3~6개월 | 15~25% |
| 6~12개월 | 30~40% |
| 12~24개월 | 45~55% |
| 24개월+ | 55~70% |

오가닉 유입은 **광고와 무관**합니다. 광고를 끈다고 줄지 않습니다. 단, SEO 기반(콘텐츠·기술 SEO·구조화 데이터)이 없으면 오가닉 유입은 시간이 지나도 쌓이지 않습니다.

---

## 변수 3 — 재방문자·단골 비율

한 번 산 고객이 다시 방문하는 비율.

### 재방문율 임계점

| 재방문율 | 사업 성격 |
|---------|----------|
| 10% 미만 | 1회성 거래 중심. 매번 신규 획득 필요 |
| 10~25% | 초기 단계. 성장 잠재력 확인 |
| 25~40% | 성숙 단계. 광고 효율 상승 |
| 40~60% | 안정 단계. 광고 의존도 낮음 |
| 60%+ | 구독·커뮤니티 성격. 광고 최소 |

재방문을 만들려면 **이메일/문자 마케팅·회원 혜택·리타겟팅**이 필요합니다. 모두 자사 도메인 + [픽셀 설치](/blog/retargeting-requires-website-pixel)가 전제입니다.

---

## 변수 4 — 콘텐츠 자산량

블로그·케이스 스터디·가이드의 축적.

### 콘텐츠 수와 오가닉 유입

| 발행 콘텐츠 수 | 월 오가닉 유입 (업종 평균) |
|---------------|--------------------------|
| 0~10편 | 50~300명 |
| 10~30편 | 300~1,500명 |
| 30~60편 | 1,500~5,000명 |
| 60~120편 | 5,000~15,000명 |
| 120편+ | 15,000명+ |

치로 블로그는 현재 60+편을 넘어섰고, 100편 이후부터는 **주제 권위(Topic Authority)**가 쌓여 새 글이 빠르게 상위 노출됩니다. 콘텐츠는 광고와 달리 **복리**로 작동합니다.

---

## 변수 5 — AI 검색 인용 (AEO)

ChatGPT·Perplexity·Gemini가 답변 출처로 당신의 사이트를 인용하는가.

### AEO 유입 특성

- 2026년 현재 전체 검색의 **25%가 AI로 이동**
- AI 인용 트래픽 CVR은 평균 SEO의 **2~3배** (초고의도)
- 한국 웹 에이전시 중 AEO 구조 구현은 **1% 미만**
- 선점 효과가 매우 큼

AEO 유입이 생기려면 다음이 필요합니다:

- llms.txt (AI 크롤러 안내 파일)
- 구조화 데이터 (JSON-LD)
- 시맨틱 HTML 구조
- robots.txt에서 GPTBot·ClaudeBot·PerplexityBot 허용

빌더로는 대부분 불가능. [코드 기반 구현](/services/seo-aeo)이 필요합니다.

## 5가지 변수의 종합 — 광고 탄력성 점수

당신의 사업 광고 탄력성을 5분 자가 진단.

### 진단 체크리스트

각 항목 예/아니오로 체크:

- [ ] 브랜드 검색량 월 1,000 이상
- [ ] SEO 오가닉 유입 비중 30% 이상
- [ ] 재방문율 25% 이상
- [ ] 발행한 블로그·콘텐츠 30편 이상
- [ ] llms.txt 파일 존재
- [ ] JSON-LD 구조화 데이터 5종 이상
- [ ] 메타 픽셀 + GA4 + Google Ads 태그 모두 설치
- [ ] 이메일 마케팅 리스트 1,000명 이상
- [ ] Core Web Vitals 전 항목 Good
- [ ] 자사 도메인으로 결제 완료

### 점수 해석

| 체크 개수 | 광고 탄력성 | 광고 중단 시 3개월 매출 유지율 |
|----------|-----------|-----------------------------|
| 0~2개 | 매우 낮음 | 5~15% |
| 3~4개 | 낮음 | 15~30% |
| 5~6개 | 중간 | 30~55% |
| 7~8개 | 높음 | 55~75% |
| 9~10개 | 매우 높음 | 75~90% |

**5개 이하**면 당신의 사업은 광고에 절대적으로 종속된 상태입니다. 광고비 변동에 매출이 즉시 흔들립니다.

---

## 탄력성 높이는 12개월 플랜

### 1~3개월차

- [홈페이지 코드 기반 구축](/services/website) 또는 리모델링
- Core Web Vitals 전 항목 Good
- 추적 코드 3종 (Meta, GA4, Google Ads) 설치
- 구조화 데이터 기본 탑재
- 첫 블로그 5편 발행

### 4~6개월차

- 블로그 주 2편 발행 (누적 20편+)
- 이메일 구독 폼 설치, 리스트 시작
- 리타겟팅 광고 개시
- llms.txt 등록

### 7~12개월차

- 블로그 누적 50~60편
- AEO 구조 완성 (구조화 데이터 10종+)
- 재방문 쿠폰·리텐션 캠페인
- 브랜드 검색량 월 1,000 돌파

1년 후 광고 탄력성 점수는 평균 **7~9개**에 도달. 광고비를 50% 줄여도 매출이 유지되는 상태.

## 자주 묻는 질문

### Q. 광고를 완전히 끊을 수도 있나요?

현실적으로 불가능합니다. 성숙 단계에서도 광고 20~30%는 **신규 고객 유입**에 필요합니다. 목표는 "광고 0%"가 아니라 "광고비 의존도 50% 이하"입니다.

### Q. B2B는 다른 기준인가요?

B2B는 재방문율이 낮은 대신 **계약 단가가 높고 LTV가 깁니다**. SEO·콘텐츠 축이 훨씬 중요하고, 광고 비중은 15~25%면 충분. 대신 콘텐츠 누적 100편+가 필수.

### Q. 빌더 사이트로는 왜 탄력성을 높이기 어려운가요?

10가지 진단 항목 중 **5~7개가 빌더로 구현 불가**이기 때문입니다. llms.txt, 페이지별 구조화 데이터, Conversion API, Core Web Vitals Good, 커스텀 추적 이벤트 등. 구조적 천장이 있습니다.

---

## 결론

광고를 끄면 매출이 0이 되는 것은 "재수가 없어서"가 아니라 **구조의 결과**입니다. 5가지 변수 중 몇 개가 작동하는지가 사업의 안정성을 결정합니다. 광고를 끊어도 굴러가는 구조를 만드는 것, 그것이 사업의 목표입니다.
