## 같은 기능, 가격은 3배 차이 — 왜일까

2026년 한국 웹 에이전시의 가격 분포:

- 저가 에이전시: 200만~300만원
- 중가 에이전시: 500만~800만원
- 고가 에이전시: 1000만~2000만원

같은 기능(SEO 최적화, 반응형 디자인, 블로그 시스템)을 하는 사이트가 200만원에도, 1000만원에도 있습니다.

10배 차이는 뭔가요. 디자인 품질. 기술 수준. 회사 규모. 모두 아닙니다.

**인건비 구조와 외주 마진의 차이입니다.**

## 200만원 에이전시의 구조

**지출:**
- 디렉터 인건비: 150만원 (10일)
- 호스팅·도메인: 20만원
- 마진: 30만원

디렉터가 직접 개발합니다. 외주 단계가 없습니다.

## 1000만원 에이전시의 구조

**지출:**
- 디렉터 컨설팅: 100만원
- 선임 개발자: 300만원
- 신입 개발자: 200만원
- 외부 프리랜서: 200만원
- 디자인 외주: 300만원
- 호스팅·도메인: 20만원
- 법인 고정비: 200만원
- 마진: 1680만원

8단계를 거칩니다. 각 단계에 마진이 누적됩니다.

## 비용을 가르는 요소

**1. 인건비 단계**

200만원: 디렉터 직접 작업 (1단계)
1000만원: 디렉터→선임→신입→외주 (4단계)

단계마다 20~30% 마진이 붙습니다.

**2. 외주 마진**

UI/UX 외주비 200만원 → 에이전시를 거쳐 청구 500만원. 마진 300만원.

**3. 법인 고정비**

대형 에이전시: 월 1500만원 고정비. 프로젝트 1개당 300만원 배분.

**4. 프로세스**

대형: 킥오프→기획→리뷰→개발→QA→리뷰→수정→배포 (8주)
소규모: 기획→개발/디자인 병렬→배포 (2주)

**5. 기술 스택**

대형: 프로젝트마다 다른 기술 선택 (개발 기간 +30%)
소규모: 검증된 1개 스택 재사용 (개발 기간 -50%)

## 1000만원이 정당한가

**정당한 경우:**
- 브랜드 전략 컨설팅 필요
- 마케팅 연동 (Google Ads·CRM 자동화)
- 엔터프라이즈급 기능 (결제·재고관리)
- 대형 조직 지원 (상장사·공공기관)

**정당하지 않은 경우:**
- 기본 홈페이지 (회사소개+블로그)
- SEO만 필요
- 디자인만 다른 경우

기본 홈페이지는 200~300만원이면 충분합니다.

## 선택의 기준

**200만~300만원:**
- 빠른 출시 필요
- 실제 성과 우선
- 이후 마케팅 예산 남기고 싶음

치로웹디자인은 이 방식입니다. [가격 정책](/pricing)을 공개합니다.

**1000만원:**
- 브랜드 전략부터 필요
- 맞춤 기술 스택 필수
- 대규모 기능 통합 필요
