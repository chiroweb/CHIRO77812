200만~300만원짜리 홈페이지 견적을 받으면 흐릿한 부분이 반드시 생깁니다. 이 가격대가 정확히 무엇을 포함하는지, 무엇은 별도인지를 계약 전에 명확히 하지 않으면 나중에 "이건 따로다"라는 말을 반복하게 됩니다.

## 200만~300만원 안에 포함되어야 할 항목

기본 웹사이트 제작비 200만~300만원에는 다음이 반드시 포함되어야 합니다.

**전략 및 설계 단계**: 시장 분석, 타겟 고객 정의, 사이트맵 구성, 와이어프레임 작성. 아임웹이나 카페24 같은 빌더는 이 단계를 건너뜁니다. 맞춤 홈페이지는 이 단계에서 웹사이트의 성패가 결정됩니다.

**디자인**: 반응형 웹 디자인(모바일·태블릿·데스크톱 최적화), UI/UX 설계, 색상·폰트·레이아웃 일관성. 페이지는 보통 5~8개(홈, 서비스, 포트폴리오, 소개, 연락처 등).

**프론트엔드 개발**: HTML, CSS, JavaScript로 만든 반응형 코드. 구글 Core Web Vitals 점수 75점 이상. SEO 기본 구조(메타 태그, 시맨틱 HTML, 구조화 데이터).

**백엔드 기본**: 연락처 폼, 간단한 문의 기능, 메일 발송 설정. 보통 Node.js나 Python 기반 서버.

**호스팅 및 배포**: Vercel, AWS, 또는 동급의 클라우드 호스팅 1년 비용 포함. 자동 SSL(https) 적용, CDN 기본 설정.

**모바일 최적화 검증**: Google PageSpeed Insights, Lighthouse 테스트 수행 및 리포트.

**기본 SEO 설정**: robots.txt, sitemap.xml, Open Graph 메타 태그, Google Search Console 연동. AEO를 위한 llms.txt 기본 설정.

**콘텐츠 최초 입력**: 홈페이지에 들어갈 텍스트, 이미지 촬영/확보. 고객이 제공한 원본 자료 기반.

## 별도 비용이 필요한 항목 (가격대 초과)

다음 항목들은 200만~300만원 범위를 벗어나거나 별도 추가비용을 요청하는 것이 정상입니다.

**커머스 기능(쇼핑몰)**: 결제 게이트웨이(PG) 연동, 재고 관리, 주문 시스템. 이 경우 최소 400만원부터 시작.

**고급 애니메이션**: Framer Motion, GSAP 기반 복잡한 인터랙션. 페이지당 +100만~150만원.

**관리자 CMS(Admin Dashboard)**: TipTap 에디터를 포함한 콘텐츠 관리 시스템. 기본 +200만원.

**다국어 지원**: 영문, 중문, 일문 등 3개 언어 이상. 언어당 +150만원.

**예약 시스템**: 병원, 미용실, 컨설팅사 등의 자동 예약 기능. +300만원.

**고급 분석 대시보드**: GA4 깊이 있는 설정, 전환 추적, 맞춤 리포트. +100만원.

**별도 사진/영상 촬영**: 기존 자료 사용이 아닌 전문 촬영. 1회 촬영당 +300만~500만원.

**브랜드 정체성 디자인**: 로고 디자인, 브랜드 가이드북. +300만~600만원 (홈페이지와 별도).

## 숨은 비용을 피하는 3가지 체크

**첫째, 페이지 수를 명확히 하세요.** "홈페이지"라는 단어는 전체 사이트를 의미하지만, 회사마다 기대치가 다릅니다. 5페이지? 10페이지? 계약서에 페이지 목록을 명시하세요.

**둘째, 이미지와 영상을 누가 준비하는지 정하세요.** 견적에 "촬영 포함"이라고 써 있지 않으면, 고객이 준비한 이미지만 사용합니다. 전문 촬영을 원하면 별도로 신청해야 합니다.

**셋째, 호스팅과 유지보수를 분리하세요.** 홈페이지 제작비와 매달 호스팅비, 유지보수비는 다릅니다. 1년 호스팅은 보통 포함되지만, 2년차부터는 누가 관리할지 미리 정하세요. 치로는 [유지보수 서비스](/services/remodeling)를 별도로 제공합니다.

## 200만~300만원짜리 홈페이지의 실제 가치

200만~300만원짜리 맞춤 홈페이지와 50만원짜리 빌더 홈페이지의 차이는 디자인이 아닙니다. 다음 세 가지입니다.

**SEO·AEO 구조**: 빌더는 URL 구조가 고정되어 있고(예: /page?id=123), 구조화 데이터를 제대로 넣을 수 없습니다. 맞춤 코드는 모든 페이지를 독립적으로 최적화할 수 있고, 구글 리치 결과에 노출될 확률이 3배 이상 높습니다.

**속도**: 아임웹, 카페24 같은 빌더 페이지의 평균 Lighthouse 점수는 50~70점입니다. 맞춤 홈페이지는 기본 80점 이상. 로딩 속도가 3배 빠르면 이탈률이 절반으로 떨어집니다.

**장기 성장성**: 빌더는 1년 후 콘텐츠를 추가할 때 구조를 바꾸기 어렵습니다. 맞춤 코드는 5년 뒤 새로운 페이지, 블로그, 전자상거래 기능을 추가할 때도 일관된 구조를 유지할 수 있습니다.

## 자주 묻는 질문

**Q: 200만~300만원에서 쇼핑몰 기능은 정말 안 되나요?**
A: 기본적인 상품 관리와 결제만 원한다면 불가능하지 않지만, 안전한 결제 처리, 재고 관리, 환불 자동화를 포함하면 최소 400만원부터 시작됩니다. 쇼핑몰은 단순 사이트와 구조가 전혀 다르기 때문입니다.

**Q: 호스팅비가 매달 추가로 드나요?**
A: 200만~300만원에 1년 호스팅이 포함됩니다. 2년차부터 매달 2만~5만원 정도의 호스팅 유지비가 발생합니다. 서버 모니터링, 보안 업데이트는 별도 [유지보수 계약](/services/remodeling)으로 진행할 수 있습니다.

**Q: 페이지를 나중에 추가하려면?**
A: 1~2개 페이지 추가는 보통 +50만~100만원입니다. 10개 이상 추가할 거라면 처음부터 큰 규모로 견적받는 것이 효율적입니다.

**Q: SEO·AEO 설정이 정말 포함되나요?**
A: 치로의 모든 200만~300만원 홈페이지에는 기본 SEO와 AEO(AI 검색 최적화)가 포함됩니다. robots.txt, llms.txt, 구조화 데이터, Core Web Vitals 최적화가 모두 기본 사항입니다.
