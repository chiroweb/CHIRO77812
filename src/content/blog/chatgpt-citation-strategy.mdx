## AI 검색에서 인용되는 사이트와 무시되는 사이트의 차이

2026년 현재 ChatGPT, Perplexity, 구글 AI Overview 같은 AI 검색 엔진이 전체 검색 트래픽에서 차지하는 비중이 30%를 넘었습니다. 같은 질문에 답하더라도 어떤 사이트는 출처로 인용되고 어떤 사이트는 무시됩니다.

AI 검색 최적화(AEO, Answer Engine Optimization)는 AI 검색 엔진이 답변을 생성할 때 내 사이트를 출처로 선택하도록 만드는 작업입니다. 전통적인 SEO와 원리가 다르고, 필요한 기술 세팅도 다릅니다.

---

## AI 검색 엔진이 출처를 선택하는 기준

ChatGPT와 Perplexity가 출처를 선택하는 방식은 공개된 알고리즘이 없습니다. 하지만 실제 인용 패턴을 분석하면 공통적인 기준이 나타납니다.

### 1. 콘텐츠의 구체성과 완결성

AI는 질문에 대한 답을 생성할 때 답변에 필요한 정보를 갖춘 페이지를 선택합니다. "SEO란 무엇인가"라는 질문에 대해, 정의만 한 줄 쓴 페이지보다 정의, 작동 방식, 실전 적용 방법까지 다룬 페이지가 선택될 가능성이 높습니다.

AI가 원하는 콘텐츠는 **질문-답변 구조**로 되어 있는 것입니다. FAQ 섹션이 있는 페이지가 없는 페이지보다 인용 확률이 높은 이유입니다.

### 2. 구조화 데이터(Schema Markup)

구조화 데이터는 검색 엔진에게 페이지 내용을 명확하게 설명하는 코드입니다. AI 검색 엔진도 이 데이터를 읽습니다.

FAQ 스키마(FAQPage)는 특히 중요합니다. 페이지에 FAQ 스키마가 삽입되어 있으면 AI가 질문과 답변을 한 쌍으로 인식하고, 그 답변을 직접 인용할 수 있습니다. Article 스키마, HowTo 스키마, LocalBusiness 스키마도 AI 인용 가능성을 높입니다.

### 3. 사이트 권위와 E-E-A-T

구글이 제시한 E-E-A-T(경험, 전문성, 권위, 신뢰) 기준은 AI 검색에서도 적용됩니다. 저자 정보, 발행 날짜, 수정 날짜, 출처 인용, 연락처 정보가 명확한 사이트가 권위 있는 출처로 인식됩니다.

AI는 신뢰할 수 없는 출처를 인용하면 답변의 신뢰도가 낮아지기 때문에, 권위 신호가 강한 사이트를 우선합니다.

### 4. 접근 가능성

AI 로봇이 사이트를 읽을 수 없으면 인용 자체가 불가능합니다. robots.txt에서 AI 로봇을 차단하거나, JavaScript 렌더링 없이는 콘텐츠가 로드되지 않는 사이트는 AI 검색에서 완전히 배제됩니다.

## Perplexity: 출처 URL을 직접 표시하는 방식

Perplexity는 ChatGPT와 다른 방식으로 출처를 표시합니다. 답변 안에 각주 번호를 붙이고, 페이지 우측에 출처 URL을 직접 나열합니다. 사용자가 각주를 클릭하면 원본 페이지로 바로 이동할 수 있습니다.

이 방식 때문에 Perplexity에서 출처로 인용되면 실제 트래픽이 발생합니다. ChatGPT가 내부 지식으로 답변을 생성하는 것과 달리, Perplexity는 실시간 웹 검색 결과를 기반으로 답변을 만들기 때문에 최신 콘텐츠가 있는 사이트에 유리합니다.

Perplexity 최적화를 위해서는 다음이 중요합니다. 콘텐츠 발행 날짜가 명확해야 합니다. 최신 정보를 다루는 콘텐츠일수록 유리합니다. 사이트 로딩이 빠르고 Perplexity 봇의 크롤링을 허용해야 합니다.

## llms.txt: AI를 위한 사이트맵

llms.txt는 AI 언어 모델이 사이트를 이해하는 데 도움을 주기 위해 만들어진 파일입니다. robots.txt가 검색 로봇에게 크롤링 규칙을 알려주는 것처럼, llms.txt는 AI에게 사이트의 핵심 정보를 요약해서 전달합니다.

llms.txt의 기본 형식입니다.

```
# 치로웹디자인

> 인천 기반 웹 디자인 스튜디오. 홈페이지 제작, SEO·AEO 최적화, 브랜딩을 제공합니다.

## 서비스 페이지
- [홈페이지 제작](https://chiroweb.co.kr/services/website)
- [SEO·AEO 서비스](https://chiroweb.co.kr/services/seo-aeo)
- [요금 안내](https://chiroweb.co.kr/pricing)

## 주요 콘텐츠
- [SEO 가이드 2026](https://chiroweb.co.kr/blog/seo-guide-2026)
- [AEO란 무엇인가](https://chiroweb.co.kr/blog/what-is-aeo)
```

이 파일을 사이트 루트(예: chiroweb.co.kr/llms.txt)에 두면 AI 로봇이 사이트 구조를 빠르게 파악합니다. 치로는 모든 [SEO·AEO 서비스](/services/seo-aeo) 프로젝트에 llms.txt 설정을 포함합니다.

## FAQ 스키마 적용 방법

FAQ 스키마는 JSON-LD 형식으로 페이지 헤더에 삽입합니다.

```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "AEO란 무엇인가요?",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "AEO(Answer Engine Optimization)는 AI 검색 엔진이 질문에 답변할 때 내 사이트를 출처로 인용하도록 최적화하는 작업입니다."
      }
    }
  ]
}
```

이 스키마가 적용된 페이지는 구글 검색 결과에서 FAQ 리치 결과로 표시되고, ChatGPT와 Perplexity가 관련 질문을 받을 때 이 Q&A 쌍을 직접 인용할 수 있습니다.

스키마 적용 후 구글 서치 콘솔의 URL 검사 도구에서 오류 여부를 반드시 확인합니다. 오류가 있는 스키마는 효과가 없습니다.

## AEO와 SEO의 차이

[AEO의 기본 개념](/blog/what-is-aeo)에서 자세히 다루고 있지만, 핵심 차이를 정리하면 다음과 같습니다.

SEO는 검색 결과 페이지에서 내 사이트 링크가 상위에 뜨도록 최적화합니다. 사용자가 링크를 클릭해야 내 사이트로 옵니다.

AEO는 AI가 질문에 답할 때 내 사이트 콘텐츠를 직접 인용하도록 최적화합니다. 사용자가 AI 답변 안에서 내 사이트를 출처로 보게 되거나, Perplexity처럼 URL을 직접 클릭해서 들어옵니다.

2026년에는 SEO와 AEO를 함께 해야 합니다. SEO만 하면 AI 검색 트래픽을 놓치고, AEO만 하면 전통 검색 트래픽을 놓칩니다.

## 치로의 AEO·SEO 통합 설계 서비스

치로는 AEO 최적화를 수동 작업이 아니라 **자동화된 구조**로 설계합니다. 사이트에 FAQ 스키마, Article 스키마, llms.txt를 체계적으로 적용하고, 새 콘텐츠가 발행될 때마다 구조화 데이터가 자동으로 생성되는 구조를 만듭니다.

AI 검색 인용 현황은 정기적으로 모니터링하고, ChatGPT와 Perplexity에서 관련 키워드를 직접 검색해 출처 포함 여부를 확인합니다. 인용이 늘어나는 패턴을 분석해 콘텐츠 전략에 반영합니다.

[SEO·AEO 서비스 페이지](/services/seo-aeo)에서 구체적인 작업 범위와 사례를 확인할 수 있습니다.

## 자주 묻는 질문

**ChatGPT가 내 사이트를 인용하는지 어떻게 확인하나요?**
ChatGPT에서 내 서비스 관련 질문을 직접 입력하고 출처를 확인합니다. ChatGPT Plus의 웹 검색 기능을 켜고 질문하면 인용 출처가 표시됩니다. Perplexity에서는 더 명확하게 확인할 수 있습니다. AI에게 "치로웹디자인에 대해 알려줘"라고 물어보는 것도 방법입니다.

**구조화 데이터를 직접 삽입할 수 있나요?**
워드프레스를 쓴다면 Yoast SEO, Rank Math 같은 플러그인이 일부 스키마를 자동으로 생성합니다. 하지만 FAQPage, HowTo, LocalBusiness 같은 스키마는 수동으로 삽입하거나 개발자가 코드에 직접 넣어야 합니다. Next.js나 커스텀 사이트에서는 JSON-LD를 페이지별로 정확하게 삽입할 수 있습니다.

**AI 검색에 노출되려면 얼마나 걸리나요?**
Perplexity는 실시간 웹 검색을 사용하기 때문에 콘텐츠 발행 후 수일 내에 반영될 수 있습니다. ChatGPT의 학습 데이터 반영은 모델 업데이트 주기에 따라 몇 달이 걸릴 수 있습니다. 단, ChatGPT에서 웹 검색 기능을 쓰는 경우에는 Perplexity와 유사하게 최신 콘텐츠가 빠르게 반영됩니다.

**llms.txt 파일이 없으면 AI 인용이 안 되나요?**
llms.txt가 없어도 AI 인용은 가능합니다. 하지만 llms.txt가 있으면 AI 로봇이 사이트 구조를 더 빠르고 정확하게 파악합니다. 경쟁 사이트가 llms.txt를 갖추고 있다면, 없는 사이트는 AI 인용 경쟁에서 불리합니다.
