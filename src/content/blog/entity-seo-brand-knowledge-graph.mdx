## AI는 문서가 아닌 개체를 기억한다

ChatGPT가 "치로웹디자인이 뭐야"라는 질문에 답할 때 페이지 하나를 인용하지 않습니다. 학습 데이터에서 "치로웹디자인"이라는 엔터티(개체)에 연결된 모든 속성을 회수합니다. 위치, 서비스, 가격, 대표자, 사례, 평판이 한 묶음으로 저장되어 있습니다. 이 구조가 지식 그래프(Knowledge Graph)입니다.

엔터티 SEO는 회사·제품·인물이 AI 지식 그래프에 정확한 속성과 함께 등록되도록 만드는 작업입니다. AEO보다 한 단계 깊은 영역입니다.

---

## 엔터티 vs 키워드 — 무엇이 다른가

키워드는 단어입니다. 엔터티는 식별 가능한 개체입니다.

| 구분 | 키워드 | 엔터티 |
|------|------|------|
| 단위 | 단어 또는 문구 | 식별 가능한 개체 |
| 예시 | "웹 에이전시" | "치로웹디자인" |
| 저장 방식 | 인덱스 매칭 | 속성 그래프 |
| 검색 결과 | 페이지 목록 | 정보 박스 + 페이지 |

구글 검색 결과 우측에 회사 정보 박스가 뜨는 것이 엔터티가 등록된 상태입니다. ChatGPT에 회사명을 물으면 즉시 정확한 답이 나오는 것도 같은 원리입니다. 박스가 없거나 ChatGPT가 "그 회사에 대한 정보가 부족합니다"라고 답하면 엔터티 미등록 상태입니다.

---

## 엔터티 등록을 위한 핵심 작업

엔터티 등록은 자동으로 일어나지 않습니다. AI가 인식할 수 있는 신호를 사이트가 명시해야 합니다.

첫째, Organization JSON-LD 스키마를 사이트 전체에 삽입합니다. 회사명, 로고, 주소, 전화번호, 대표자, 설립일, 사업 분야가 구조화 데이터로 표현됩니다.

```json
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "치로웹디자인",
  "founder": {"@type": "Person", "name": "최정원"},
  "foundingDate": "2024-04-01",
  "address": {
    "@type": "PostalAddress",
    "addressLocality": "서울",
    "addressCountry": "KR"
  },
  "sameAs": [
    "https://www.instagram.com/chiro.web",
    "https://www.threads.net/@chiro.web"
  ]
}
```

`sameAs` 필드는 엔터티 등록에서 가장 중요한 항목입니다. 동일한 회사가 운영하는 외부 프로필 URL을 연결해 AI가 같은 개체로 묶을 수 있게 합니다.

---

## sameAs — 엔터티 통합의 핵심

AI는 흩어진 정보를 어떻게 하나의 엔터티로 묶을까요. sameAs 신호와 일관된 NAP(Name, Address, Phone) 데이터를 사용합니다. 같은 회사명, 같은 주소, 같은 전화번호가 5개 이상 신뢰 소스에서 일치하면 같은 엔터티로 판단합니다.

치로웹디자인이 등록한 sameAs 후보 목록입니다.

- 인스타그램 공식 계정
- 스레드 공식 계정
- 링크드인 회사 페이지
- 카카오톡 채널
- 네이버 플레이스
- 구글 비즈니스 프로필
- 위키데이터 항목

7개 외부 신호가 일치하면 AI 지식 그래프에 엔터티가 등록될 확률이 매우 높아집니다.

---

## 위키데이터 — AI 학습 데이터의 핵심 소스

ChatGPT, Claude, Gemini 모두 위키데이터(Wikidata)를 학습 데이터의 권위 소스로 사용합니다. 회사명을 위키데이터에 등록하면 엔터티 신호가 강력해집니다.

위키데이터 항목 작성 조건은 다음과 같습니다. 회사가 실제 운영 중이어야 합니다. 외부 신뢰 소스(언론 보도, 위키피디아 인용 등)가 1개 이상 있어야 합니다. 항목 작성 후 위키데이터 커뮤니티 검토를 거칩니다.

처음에는 등록이 거부될 수 있습니다. 외부 보도자료 또는 산업 매체 인용이 누적되면 등록 가능성이 올라갑니다.

## 엔터티 검증 방법

회사 엔터티가 AI에 등록되었는지 확인하는 4가지 검증입니다.

1. ChatGPT에 "회사명에 대해 설명해줘" 입력. 정확한 답변이 나오면 등록 상태.
2. Perplexity에 같은 질문 입력. 답변과 함께 자사 도메인이 출처에 포함되는지 확인.
3. 구글 검색에서 회사명 검색. 우측에 정보 박스가 나타나는지 확인.
4. Knowledge Graph API 호출. Google Cloud Console에서 직접 조회 가능.

[AI 검색 가시성 점검](/blog/ai-search-visibility-check)에서 검증 도구 사용법을 다룹니다.

## 엔터티 강화 6개월 로드맵

치로웹디자인이 사용한 6개월 엔터티 강화 작업 순서입니다.

월 1: Organization 스키마 + sameAs 5개 등록.
월 2: 구글 비즈니스 프로필, 네이버 플레이스 신청과 검증.
월 3: 위키데이터 항목 작성 시도.
월 4: 산업 매체 1~2곳 기고 또는 인터뷰.
월 5: 보도자료 1건 배포 + 외부 백링크 5개 확보.
월 6: 모든 신호 통합 점검 + ChatGPT 답변 모니터링.

[AEO·SEO 통합 설계 서비스](/services/seo-aeo)는 1~3개월차 작업을 표준 작업으로 포함합니다.

## 자주 묻는 질문

**스타트업이나 신규 회사도 엔터티 등록이 가능한가요?**
가능합니다. 다만 시간이 필요합니다. 도메인 권위와 외부 인용이 누적되어야 AI가 신뢰 가능한 엔터티로 판단합니다. 6~12개월의 일관된 활동이 필요합니다.

**위키피디아 등록이 꼭 필요한가요?**
필수는 아닙니다. 위키데이터(Wikidata) 등록이 더 우선입니다. 위키피디아는 진입 장벽이 매우 높지만 위키데이터는 검증된 외부 소스만 있어도 등록 가능합니다.

**엔터티가 등록되면 AEO 효과가 보장되나요?**
엔터티 등록은 출발선일 뿐입니다. 등록 후에도 콘텐츠 품질, 백링크, 신선도가 함께 갖춰져야 인용이 일어납니다. 엔터티는 AI가 "당신이 누군지 안다"는 의미지, "당신을 추천한다"는 의미가 아닙니다.
