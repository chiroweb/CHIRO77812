## 200만원과 300만원의 차이는 기술이 아니다

왜 어떤 에이전시는 200만원, 어떤 곳은 300만원일까요. 같은 기술을 써도 가격이 다른 이유는 페이지 수, 포함 기능, 콘텐츠 준비도입니다. 이 3가지 변수가 최종 가격을 결정합니다. [풀스택 홈페이지의 기술적 가치](/blog/fullstack-vs-builder-ai-search-strength)를 먼저 이해하면 가격이 왜 정해지는지 명확해집니다.

치로웹디자인의 기본 가격은 200만원입니다. 여기서 300만원대로 오르는 이유를 명확하게 분해하면, 당신이 최고의 가성비로 홈페이지를 의뢰할 수 있습니다.

## 변수 1 — 페이지 수 (가장 큰 영향)

**200만원 기본 구성:** 메인(1) + 소개(1) + 서비스(3~4) + 블로그(2) = 평균 7~8페이지.

**250만원대:** 위 구성 + 서비스 상세 페이지 3개 추가 = 11~12페이지.

**300만원대:** 위 구성 + 팀 소개(1) + 가격 페이지(1) + 자주 묻는 질문(1) + 고객 사례(3~5) = 16~18페이지.

각 페이지는 단순히 "텍스트 모음"이 아닙니다. 각 페이지는:

- 독립적인 시맨틱 마크업 구조
- 그 페이지만의 JSON-LD 스키마
- SEO 최적화 (메타 제목, 설명, 키워드)
- 내부 링크 전략

이 모든 작업이 페이지 수에 정비례합니다. 페이지당 평균 3~4시간의 작업이 필요합니다. 8페이지는 24~32시간, 16페이지는 48~64시간입니다. 시간 + 관리 비용으로 변환하면 이 차이가 100만원이 됩니다.

## 변수 2 — 기능 (비용 변동폭: 20~50만원)

기본 기능(예약, 문의 폼, 뉴스레터)는 모든 200만원대 홈페이지에 포함됩니다. 추가 기능이 필요하면 가격이 올라갑니다:

| 기능 | 비용 추가 | 예시 |
|------|---------|------|
| 결제 시스템 (Stripe) | 0~10만원* | 디지털 상품 판매, 컨설팅 결제 |
| 회원 가입/로그인 | 15~20만원 | 고객 대시보드 (이전 상담 기록 조회) |
| 예약 시스템 (Google Calendar 연동) | 10~15만원 | 병원, 코칭, 상담 예약 |
| 실시간 채팅 (Intercom) | 0~5만원* | AI 챗봇 + 수동 상담 |
| 다국어 지원 (3개 언어) | 20~30만원 | 영어/중국어/일본어 자동 번역 |
| 동적 콘텐츠 관리 (CMS) | 10~20만원 | 직원이 직접 콘텐츠 수정 가능 |

*마크는 Vercel의 free tier에서 처리 가능, 추가 비용 거의 없음.

예를 들어 병원 홈페이지라면:
- 200만원: 기본 홈페이지 + 진료 예약 폼
- 250만원: 위 + Google Calendar 예약 시스템 (자동 확인/취소)
- 270만원: 위 + 회원 가입 (진료 기록 조회)
- 300만원: 위 + 다국어 지원 (영어)

## 변수 3 — 콘텐츠 준비 상태 (생각보다 큰 영향)

에이전시는 기술만 제공합니다. 콘텐츠는 고객이 준비합니다. 그런데 대부분의 고객은 "어떤 콘텐츠가 필요한지" 모릅니다. 여기서 시간이 낭비되고, 그 시간이 비용으로 전환됩니다.

**콘텐츠 준비도 A (비용 추가 0):**
- 회사 소개 텍스트 준비됨 (200~300자)
- 서비스 설명 정리됨 (각 100~150자)
- 로고·색상 가이드 있음
- 팀 사진 3~5장 준비됨
- 고객 사례 2~3개 있음

결과: 200만원대로 제작 가능. 제작 기간 7~9일.

**콘텐츠 준비도 B (비용 추가 30~50만원):**
- 회사 소개는 있지만 SEO 키워드 없음
- 서비스 설명이 모호함 ("품질 높은 서비스" 같은 추상어)
- 로고는 있지만 브랜드 가이드 없음
- 팀 사진 없음 (촬영 필요 또는 기존 사진 활용)
- 고객 사례 콘텐츠 부분적

에이전시는 콘텐츠를 "인터뷰"해서 정리해야 합니다. 이 인터뷰 시간이 8~12시간 추가됩니다. 비용으로는 30~50만원. 제작 기간 12~15일.

**콘텐츠 준비도 C (비용 추가 50~100만원 + 기간 연장):**
- 회사 소개가 없음 (처음부터 작성 필요)
- 서비스 종류가 명확하지 않음 (정리 필요)
- 로고 없음 (디자인 필요)
- 팀 사진 없음, 촬영도 안 함 (텍스트만 또는 새 촬영)
- 고객 사례 콘텐츠 없음

이 경우, 에이전시는 단순 코더가 아니라 "카피라이터 + 인터뷰어 + 기획자"가 됩니다. 작업 기간이 20일 이상으로 늘어납니다. 비용은 300만원을 넘어갑니다. 치로웹디자인은 이 구간에서는 별도 상담이 필요하다고 안내합니다.

## 최종 가격 공식

최종 가격 = 기본 200만원 + (페이지 수 - 8) × 12~15만원 + 추가 기능 + 콘텐츠 준비도

**예시:**
- 변호사: 200만원 + 90만원 + 15만원 = 305만원 (14페이지)
- 제조업: 200만원 + 24만원 + 30만원 = 254만원 (10페이지)

## 최고의 가성비를 얻는 방법

**콘텐츠 미리 준비:** 회사 소개·서비스 설명·로고·팀 사진을 미리 준비하면 30~50만원을 절감합니다.

**페이지 우선순위:** 메인+소개+서비스 3개+문의(200만원)부터 시작해, 단계별로 추가합니다. [전문직 홈페이지의 가성비 전략](/blog/professional-services-website-value-200-300)을 참고하면 분야별 최적 구성을 찾을 수 있습니다.

**기능은 필수만:** 예약·결제 등은 정말 필요한가 판단하세요.

## 자주 묻는 질문

**페이지를 추가하면서 가격을 내릴 수는 없나요?**

페이지는 시간에 정비례합니다. 다만 "장기 계약"으로 월 1페이지씩 6개월 추가 시 260만원(1페이지당 15만원)으로 정할 수 있습니다.

**콘텐츠 작성까지 에이전시가 해주나요?**

치로웹디자인은 콘텐츠 인터뷰는 무료로 돕지만, 전문 카피라이팅은 별도 비용입니다. 기본은 고객 텍스트를 구조화하는 수준입니다.

**200만원대 홈페이지는 2년 후 유지비가 많이 드나요?**

아닙니다. 월 유지비가 0~5만원입니다(호스팅·DB). 빌더는 매달 15~30만원이 필요합니다. 2년 후 누적 비용으로 풀스택이 훨씬 저렴합니다.

**이미 빌더 사이트가 있는데 옮기면 얼마인가요?**

"리모델링" 패키지로 150만~180만원대, 10일 소요입니다. 도메인 유지, 구조만 새 설계합니다.
