## 제작비 300만원짜리 사이트의 1년 실제 비용

제작비 300만원에 계약했지만 1년 뒤 총비용이 550만원이 되는 경우가 흔합니다. 제작비 이외의 항목을 사전에 계산하지 않았기 때문입니다.

---

## 연간 총비용 계산표

| 항목 | 빌더 (아임웹 기준) | 코드 기반 (치로 기준) |
|------|-----------------|-------------------|
| 제작비 (1회성) | 0원 (직접 제작) | 300~800만원 |
| 월 플랫폼 이용료 | 3~8만원 × 12 = 36~96만원 | 0원 |
| 호스팅 | 플랫폼 포함 | 연 12~24만원 |
| 도메인 | 연 1~2만원 | 연 1~2만원 |
| SSL | 플랫폼 포함 | 무료 (Let's Encrypt) |
| 유지보수 | 별도 계약 시 월 5~15만원 | 치로 포함 |
| 콘텐츠 업데이트 | 직접 또는 월 10~30만원 | 직접 가능 |
| **1년 합계** | **37~300만원 + 제작 시간** | **제작비 + 13~26만원** |

---

## 빌더의 숨은 비용: 시간

아임웹을 직접 만들면 초기 제작비가 없습니다. 하지만 사업자가 직접 사이트를 만드는 데 드는 시간이 비용입니다. 평균적으로 비전문가가 아임웹으로 제대로 된 사이트를 만드는 데 40~80시간이 소요됩니다. 시간당 기회비용을 5만원으로 잡으면 200~400만원입니다.

---

## 3년 총비용 비교

빌더를 쓰면서 광고비로 월 30만원을 지출하는 사업체와, 코드 기반 사이트로 검색 유입을 만드는 사업체의 3년 비용 차이는 상당합니다.

- 빌더 + 광고: 플랫폼 비용 + 광고비 36만원 × 36개월 = 1,296만원 이상
- 코드 기반 + SEO: 제작비 1회 + 유지 연 26만원 × 3년 = 제작비 + 78만원

광고를 줄이고 검색 유입을 늘리는 구조로 전환하면 3년 누적 비용이 역전됩니다. 검색 최적화된 코드 기반 사이트가 언제 손익분기점을 넘는지는 업종과 키워드 경쟁도에 따라 다르지만, 대부분 12~18개월 내에 역전됩니다.

## 치로의 가격 구조

치로는 제작비 외에 추가 항목이 없습니다. 호스팅, SSL, 도메인 설정, 기본 SEO 구조가 제작비에 포함됩니다. 유지보수 계약을 강제하지 않습니다.

[치로웹디자인 가격 안내](/pricing)에서 전체 포함 항목을 확인할 수 있습니다. 현재 사이트의 비용 구조가 적절한지 판단이 필요하다면 [홈페이지 제작 서비스](/services/website) 페이지에서 상담할 수 있습니다.

## 자주 묻는 질문

**호스팅 비용을 줄이는 방법이 있나요?**
Vercel, Cloudflare Pages 같은 플랫폼은 소규모 사이트 기준 무료 호스팅을 제공합니다. 치로는 이 플랫폼을 기본으로 사용해 별도 호스팅 비용이 없습니다.

**콘텐츠 업데이트는 직접 할 수 있나요?**
치로가 만든 사이트는 CMS(관리자 패널)가 포함됩니다. 블로그 글, 포트폴리오, 서비스 소개 등 콘텐츠를 직접 수정할 수 있습니다.

**제작 후 사이트 소유권은 누구에게 있나요?**
치로는 완성된 사이트의 코드와 도메인, 호스팅 계정 모두 고객에게 이전합니다. 계약 종료 후에도 독립적으로 운영 가능합니다.
